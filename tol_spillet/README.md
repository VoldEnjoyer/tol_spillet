# 🍌 Sigma Banan Mafia Quest 🍌

Det ultimate spillet hvor du må bekjempe mafia-banditter med ballekicks for å ta ned den legendariske **Sigma Sigma Boy Banan**!

## 🎮 Hvordan spille

### Kontroller:
- **WASD** - Beveg spilleren
- **E** - Snakk med NPCs / Få quests
- **Spacebar** - Vanlig angrep
- **K** - **BALLEKICK!** (Den ultimate sigmateknikken) 🦶💥

### Spillmål:
1. Gjør 3 quests for å bli sterk nok
2. Slå alle mafia-medlemmene med ballekicks for å stunne dem
3. Vekk opp Sigma Banan Boss
4. Bekjempe den ultimate boss med dine ballekick-ferdigheter!
